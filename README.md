# Simulador de Orçamento Freelancer

Este projeto é um simulador de orçamento interativo, onde o usuário pode selecionar serviços e visualizar o valor total em tempo real, simulando uma experiência de compra.

## Funcionalidades

- Adicionar serviços ao orçamento  
- Cálculo automático do valor total  
- Lista de serviços selecionados  
- Botão de finalizar orçamento  
- Botão de resetar orçamento  
- Interface simples e responsiva  

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Como usar

1. Abra o arquivo `index.html` no navegador  
2. Clique nos serviços para adicioná-los ao orçamento  
3. Veja o total sendo atualizado automaticamente  
4. Clique em **Finalizar** para simular a compra  
5. Clique em **Resetar** para limpar tudo  

## Estrutura

- `index.html`  
- `style.css`  
- `script.js`
